We need Blend in this project

Open this project in Blend Studio
Make sure you select on Object and Timeline tab is visible 

On Solution Explorer tab, select CustomerListView.xaml

Select Assets tab

Then select Behaviors.

On Object and Timeline,select the UserControl

Go back to Assets tab, Behaviors and double click on CallMethodAction. As you will note on the xaml, new code will be added.

Save it and open again the project in Visual Studio
